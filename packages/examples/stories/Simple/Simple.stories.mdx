# Simple Component Example

## MDX

Importing `CssPropsBlock` and configuring directly in MDX.

```tsx
import { CssPropsBlock } from "@ljcl/storybook-addon-cssprops";
```

<CssPropsBlock
  customProperties={{
    ...SimpleStories.parameters.cssprops,
    "css-custom-property-1": {
      value: "orange",
      description: "Defined in an MDX file!",
    },
  }}
/>
