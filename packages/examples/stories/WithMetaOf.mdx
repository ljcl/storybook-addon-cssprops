# MDX with `<Meta of={Stories}/>`

When populating the Meta component with stories, `CssPropsBlock` will find and use associated CSS Custom Properties.

```tsx
import * as SimpleStories from "./Simple.stories";
import { CssPropsBlock } from "@ljcl/storybook-addon-cssprops";

<Meta of={SimpleStories} />

<CssPropsBlock />

<Canvas of={SimpleStories.DefaultStory}></Canvas>
```
